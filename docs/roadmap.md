# Roadmap

## Phase 1

- FastAPI 백엔드 스캐폴드
- Shot / Task / Version 모델
- Google Sheets ShotMaster 동기화
- 기본 샷 조회 및 상태 변경 API

## Phase 2

- `PipelineRun`, `PipelineStepRun` 모델 추가
- 샷 실행 API 추가
- Worker polling 또는 queue consumer 작성
- 샷 리스트용 상태 집계 API 작성

## Phase 3

- Maya connector
- Blender connector
- After Effects connector
- 단계별 로그/썸네일/결과물 저장

## Phase 4

- 웹 UI
- 프로젝트 선택 화면
- 샷 리스트 및 필터
- 실행 이력/실패 원인 화면

## Phase 5

- NAS 배포 자동화
- 계정/권한
- 알림
- 에러 복구 흐름
